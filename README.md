# 1Win Sri Lanka Landing Page (Affiliate)

මෙය 1Win සබැඳි ක්‍රීඩා අඩවියේ ශ්‍රී ලංකාවේ පාරිභෝගිකයින් ඉලක්ක කර නිර්මාණය කරන ලද, **සිංහල සහ ඉංග්‍රීසි භාෂා දෙකෙන්ම** ක්‍රියා කරන අලෙවිකරණ ගොඩබිම් පිටුවක් (Landing Page) වේ.

## 🚀 විශේෂාංග (Features)

- 🌐 **භාෂා දෙකක සහය**: එක් ක්ලික් කිරීමකින් සිංහල සහ ඉංග්‍රීසි භාෂා අතර මාරු වීම.
- 📱 **මොබයිල් රෙස්පොන්සිව්**: සියලුම උපාංග සඳහා (දුරකථන, ටැබ්ලට්, පරිගණක) ප්‍රශස්ත කර ඇත.
- ⏱️ **කාල සීමා ගණකය (Countdown Timer)**: සීමිත කාල ප්‍රවර්ධනයක් පෙන්වීමට.
- 🎁 **ප්‍රොමෝ කෝඩ් පිටපත් කිරීම**: "WINLANKA" කෝඩ් එක එබීමෙන් කෙලින්ම ක්ලිප්බෝඩ් එකට පිටපත් වේ.
- 🚪 **Exit-Intent Popup**: පිටුවෙන් පිටවීමට උත්සාහ කරන විට විශේෂ බෝනස් ප්‍රවර්ධනයක් පෙන්වයි.
- 🍪 **කුකී බැනරය (Cookies Banner)**: භාවිතා කරන්නන්ගේ කැමැත්ත පිළිබඳව දැනුම් දීම.
- 📊 **සජීවී අමුත්තන් ගණකය (Live Visitor Counter)**: අඩවියේ සිටින තත්කාලීන අමුත්තන් ගණන පෙන්වයි.
- 🎡 **ස්ලයිඩරය (Auto-slider)**: ක්‍රීඩා සහ බෝනස් පින්තූර ස්වයංක්‍රීයව භ්‍රමණය වේ.
- 🔄 **FAQ Accordion**: නිතර අසන ප්‍රශ්න කොටස සංක්ෂිප්තව හා විස්තීර්ණව පෙන්වීමේ පහසුකම.

## 🛠️ භාවිතා කළ තාක්ෂණයන් (Technologies Used)

- **HTML5** - ව්‍යුහය
- **CSS3** - නිර්මාණ ශෛලීන් (Tailwind CSS මගින් පිටතින් ලබාගෙන ඇත)
- **JavaScript (Vanilla ES6)** - සියලුම ක්‍රියාකාරීත්වයන්
- **Font Awesome 6** - අයිකන සඳහා
- **Google Fonts (Noto Sans Sinhala)** - සිංහල අකුරු සඳහා
- **Schema.org JSON-LD** - SEO සඳහා ව්‍යුහගත දත්ත
